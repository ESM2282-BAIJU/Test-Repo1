## Hello World Scripts

This repository includes two simple Hello World scripts.

- `HelloWorld.js`: Node.js script. Run with `node HelloWorld.js [name]` or make it executable and run `./HelloWorld.js [name]`.
- `hello.py`: Python 3 script. Run with `python3 hello.py [name]` or make it executable and run `./hello.py [name]`.

Examples:

```bash
# Node.js
node HelloWorld.js
node HelloWorld.js Alice

# Python
python3 hello.py
python3 hello.py Bob
```
